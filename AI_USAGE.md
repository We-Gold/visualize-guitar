# AI Usage

My AI usage follows the guidelines outlined in the course syllabus.

GitHub Copilot autocomplete was used throughout the project, including to make documentation for the functions.

Agentic AI was used for parts of the following features/actions:

- Rewrite processing code to merge MIDI and GP5 instead of just parsing GP5
- Refactor audio controller and guitar sampler
- Get positions and dimensions from Guitar SVG file
    - Animate guitar strings
    - Animate fret finger position transitions
- Get positions and dimensions from Selector SVG file
    - Animate selector
- Migrate plots from D3 to Canvas
- Create initial editor component
- Create notes (midi) viewer component
